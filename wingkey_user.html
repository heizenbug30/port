2021 (주)글로벌폰유학 · 안드로이드 · 관리자

작업범위

  • 브랜디드
  • 전 페이지 디자인 및 프로토타이핑

윙키는 하이브리드 앱으로 해외 원어민과 pc 태블릿 모바일 여러 기기에서 원어민 강사 매칭부터 온라인 수업까지 올인원으로 진행할 수 있도록 돕는 서비스 입니다.

주 타겟층은 2-30 학생 및 직장인이며, 관심사나 취미가 같은 원어민을 매칭할 수 있다는 점이 핵심 포인트로 중국어 서비스를 먼저 오픈하였습니다.

개발단계에 참여하여 런칭후 탑재될 컨텐츠 디자인에 먼저 집중했으며, 개발완료 후 내부테스트를 통해 오류를 수정하고 최종적으로 개선할 점을 찾았습니다.

1추가적인 기초조사 : 국내 외국어 회화 수요 /

글로벌 시대, 국내 외국어 회화 수요는 늘어나고 있지만, 동기부여 장치가 없어서 대부분 중도 포기

외국어 회화 학습자 학생의 경우 튜터 품질(학력 등)이 가장중요 / 직장인의 경우 자율적인 시간변경, 가격이 가장중요하다고 응답

2.윙키 서비스 조사 및 경쟁사 조사

윙키 개발전 푸다오바라는 브랜드로 동일한 서비스를 출시했었습니다. 성과측정을 위해 푸다오바 데이터의 구매율 재구매율등을 조사하였습니다

동종업계는 대표적으로 YBM 문정아중국어 우리차이나가 있으며, 1:1과외 방식은 동일하나 별도 앱이 없는경우가 다수였습니다.

유사업계중 경쟁사중 링글은 2-30대 학생/직장인 타겟으로 튜터의 전공 혹은 전문분야에 대한 정보검색이 용이하며 수업시간이 제한적임 자체온라인 강의실이 있음

취미나 관심사를 강조한 ui는 00으로 컨셉은 비슷하나 자체 온ㄹ라인 강의실없음

유사시장 : 스픽 / ai 외국어 회화이지만 레벨 테스트 부터 수업 복습앍람까지 다채로운 기능이 있음

4. 테스터 모집 인터뷰

사용자가 없는상태에서 진행할 수 ㅇ씨는 테스터 모집 및 데이터 조사를 실시

가격 및 튜터 품질은 대체적으로 만족

사용성은 꽝

4.내부 관계자 인터뷰

2-30대 타겟을 1차로 어린이 중장년 층으로 확장. 시급한 오류를 수정후 윙키만의 핵심기능을 부각시킬수 있어야 함

-학생들이 선호하는 품질관리 그리고 복습 시스템점건이 필요함

-직장인을 위한 10분 수업부터 자유로운 시간변경이 가능해야함

-윙키 핵심기능에 적합한 ui로 개선해야함

-마케팅 측정을 위한 ga연결 / 수업 실시간 모니터링이 가능해야함

-글로벌 사용자를 고려한 디자인이여야 함

-페이지/ 버튼레이블 통일 -가격 0원으로 표시되는 문제 -> 결제 화폐 선택안할 경우 표시안됨 -> 정책상 반드시 선택해야 하는 옵션이므로 회원가입시 필수 입력사항으로 전환 + 비회원일 경우 "회원전용" 이라고 표시 > 로그인 후 다시돌아오게 할것인가?

3.강의실입장

-강의실 입장버튼 이슈 : 그외라인 중국라인 이라는 애매한 레이블 -> 사용자 국가에 맞추어 각기 다른서버의 강의실로 자동 입장되는것이 바람직 / 임시방편으로 사용하려면 사용자가 굳이 알 필요 없는 정보이므로 사용자입장에서 맞게 버튼명변경 필요 -내수업 사용자 전환 시점설계 누락으로 인한 치명적인 이슈 (학생과 튜터를 1개의 계정으로 이용하는 하이브리드앱으로 가장 중요한 부분임) -> 토글버튼 + 회원종류에 맞는 페이지 노출 (전반적으로 개선필요) https://drive.google.com/file/d/182lJXgO2KlIBgiHo-L0fKegzD3P7j--5/view?usp=drive_link -내수업 접근자체가 어려움 > 페이지 개편이 어려운 상황으로 pc화면에서 바로가기 버튼 임시 추가

4. 수업진행도 확인(출석부) > 수업완료

-출석부 그룹수업과 개인수업 동일한 양식으로 사용 발생되는 이슈 -출석부 수업선택 좌우 슬라이드 오류 드롭다운으로 변경 / 혹은 페이지개념으로 변경 2안 작성 -수업신청부터 완료까지 필요한 모든 기능들이 여러 페이지에 흩뿌려져있음 수업기록 피드백 출석상태를 확인하려면 출석부에서 다시 수업을 찾아야하는 구조 (사용성문제) > 출석부 버튼을 내수업과 연결 필수
